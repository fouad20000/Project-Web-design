As a student, I like combining what I learn in class with real projects. Projects like this help me practice problem-solving, UI design, and clean coding.

I chose a Dota 2 themed project because it’s something I enjoy and understand. Turning the item shop into a web app makes the assignment more personal and shows that I can connect my interests as a gamer with my work as a student.
